Pantalla de Splash

La pantalla de splash es una pantalla de presentación que se muestra mientras la PWA se está cargando. Es una especie de "pantalla de bienvenida" que aparece antes de que la aplicación esté completamente cargada. Su propósito es proporcionar una transición suave entre el inicio de la aplicación y la carga completa, brindando a los usuarios una indicación visual de que la aplicación está en proceso de carga. La pantalla de splash generalmente muestra el logotipo de la aplicación o algún otro gráfico representativo.


HTML

Este código HTML crea una estructura para mostrar un indicador visual de carga en el cuerpo de la página, con una imagen y un conjunto de puntos animados. La clase hidden inicialmente oculta el cuerpo de la página, y se usa JavaScript para quitar esa clase y mostrar el contenido una vez que la página haya cargado completamente.


JavaScript

El código se asegura de ocultar el elemento con id 'onLoad' y de revelar el cuerpo de la página después de que todos los recursos se hayan cargado. Esto puede ser útil, ya que el elemento de carga ('onLoad') se muestra mientras la página se está cargando, y se oculta una vez que todo ha terminado de cargarse.


Estilos

Estos estilos crean una interfaz visual de carga con un fondo de color, una imagen de carga y un conjunto de puntos animados que proporcionan un indicador visual de que la página está procesando o cargando datos.

Pantalla de Home

La página de inicio es la primera pantalla que ve un usuario después de que la PWA se ha cargado completamente. Es la interfaz principal de la aplicación. Esta página suele contener los elementos clave de la aplicación y proporciona la funcionalidad principal. Puede incluir menús de navegación, contenido principal, botones de acción, etc. La idea es que la página de inicio sea la puerta de entrada principal a la experiencia de la aplicación web progresiva.


Nuestra pantalla de inicio contiene una barra de navegación y el muero.
En la barra de navegación, del lado izquierdo se encuentra el nombre la aplicación web. En el centro se encuentran tres botones los cuales son inicio, (la cual te direcciona a la pantalla de inicio) contacto (la cual contiene un correo para ponerse en contacto con los administradores) y reportes (a los cuales solo tienen acceso los administradores). Finalmente, del lado derecho se encuentra el botón salir o iniciar (los cuales funcionan para hacer loggin o logout).
En el muro se encuentra una card en la cual el usuario puede realizar una publicación pudiendo poner el título, el contenido y agregar una imagen a su publicación. El resto son publicaciones de los usuarios almacenados en una base de datos y siendo solicitados a la aplicación web de forma asíncrona.
