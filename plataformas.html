Plataformas y herramientas utilizadas.

Justificación de las plataformas y herramientas utilizadas:

Para el desarrollo la aplicación se decidió utilizar el lenguaje de programación Python con el framework de desarrollo Flask, esto se debe a que encontramos en este una serie de características y beneficios que creemos que se adaptan a los requerimientos del proyecto.

Flexibilidad y rendimiento: Flask es un framework web ligero y flexible, diseñado principalmente para construir aplicaciones web pequeñas y medianas, una de las características que lo destaca sobre otros frameworks de Python es su eficiencia y velocidad, ya que se ejecuta rápidamente al utilizar solo los componentes esenciales necesarios para construir una aplicación web.

Escalabilidad y compatibilidad: Flask ofrece una base sólida para el crecimiento de la aplicación ya que permite organizar el código en módulos y extensiones, lo que facilita la gestión y la escalabilidad a medida que el proyecto crece y requiere más funcionalidades, al mismo tiempo, Flask también es compatible con el uso de servicios y herramientas externas para manejar aspectos como el almacenamiento en caché y la gestión de bases de datos.

Seguridad: Flask proporciona características básicas de seguridad como soporte para protección contra ataques CSRF (Cross-Site Request Forgery) y XSS (Cross- Site Scripting) que facilitan la implementación de mecanismos de seguridad contra ciertas vulnerabilidades en una aplicación.

Justificación de Git y Git-Hub:

Estamos utilizando Git y GitHub en el desarrollo de nuestro proyecto del foro universitario "Castor Senpai" ya que tiene bastantes ventajas como la colaboración y gestión del proyecto.

Control de Versiones(Git):

Git es un sistema de control de versiones distribuido ampliamente utilizado. Permite realizar un seguimiento preciso de los cambios en el código fuente a lo largo del tiempo. Cada modificación es registrada, lo que facilita la identificación y resolución de problemas, así como la reversión a versiones anteriores si es necesario.

Colaboración y Gestión de Código (GitHub):

GitHub proporciona una plataforma centralizada para el alojamiento de repositorios Git, lo que facilita la colaboración entre desarrolladores. Permite a varios miembros del equipo trabajar simultáneamente en el código, fusionar cambios de manera eficiente y gestionar problemas y solicitudes de extracción de manera estructurada.

Revisión de Código:

GitHub facilita la revisión de código a través de solicitudes de extracción. Los cambios propuestos pueden ser revisados, comentados y discutidos antes de su fusión en la rama principal del código. Esto asegura la calidad del código y proporciona un mecanismo transparente para la aprobación de nuevas características o correcciones.

Justificación de mysql:

Se decidio usar MySQL como sistema de gestion de nuestra base de datos ya que es capaz de manejar grandes conjuntos de datos, por lo que es importante para un sistema universitario que puede acumular una cantidad significativa de información a medida que se utilizan y generan datos a lo largo del cuatrimestre ademas proporciona opciones de escabilidad para crecer con las necesidades del proyecto en caso de que se decida hacer actualizaciones para evolucionar el software. Puede ser configurado para manejar un mayor número de transacciones y usuarios a medida que el proyecto Castor Senpai evoluciona y aumenta su base de usuarios.

También MySQL es compatible con un amplio conjunto de herramientas y lenguajes de programación. Esto permite una fácil integración con otras tecnologías y facilita el desarrollo de aplicaciones en un entorno diverso y ofrece funciones avanzadas de seguridad, como la gestión de usuarios y privilegios, encriptación de datos. Estas características son esenciales para proteger la confidencialidad y la integridad de los datos sensibles de los estudiantes y profesores.

Justificación de bibliotecas:

bcrypt: Se utilizo esta biblioteca de python para poder realizar la encriptación de las contraseñas ya que es una biblioteca de Python que se utiliza para el cifrado seguro de contraseñas. Funciona mediante la aplicación de un algoritmo de hash lento y deliberadamente costoso, diseñado específicamente para ser resistente a ataques de fuerza bruta.

Pillow: Pillow es una biblioteca de procesamiento de imágenes en Python que se utiliza para trabajar con imágenes en diferentes formatos. Permite abrir, manipular, y guardar imágenes. Utilizamos esta biblioteca para limitar a los usuarios a usar solo imagenes dentro de sus publicaciones.

mysql-connector-python: Usamos esta biblioteca para la conexión de nuestra base de datos con el proyecto "Castor Sempai" ya que es un conector oficial de MySQL para Python que facilita la interacción con bases de datos MySQL desde aplicaciones Python. Permite realizar operaciones como la ejecución de consultas SQL, inserción y actualización de datos en la base de datos MySQL.

Parametros de Configuración:
  • Instalación de MySQL:
    Primero, se debe instalar el servidor de MySQL en el servidor. Esto varía según el sistema operativo.
    sudo apt-get install mysql-server
  • Instalación de Python:
    La mayoría de los servidores Unix ya tienen Python instalado. Asegúrarse de que Python esté instalado y disponible. Se puede verificar la versión con el siguiente comando:
    python –versión
  • Instalación de MySQL Python Connector:
    El conector MySQL para Python se puede instalar mediante pip, que es el administrador de paquetes de Python. Ejecutar el siguiente comando para instalarlo:
    pip install mysql-connector-python
  • Instalación de Flask:
    Flask es un marco de desarrollo web para Python. Puede instalarse también con pip:
    pip install Flask
  • Instalación de Bcrypt:
    Bcrypt es una biblioteca de Python que se utiliza para el hashing seguro de contraseñas. Instálarlo con pip.
    pip install bcrypt
  • Instalación de Pillow:
    Pillow es una biblioteca de código abierto que es una bifurcación (fork) de la biblioteca Python Imaging Library (PIL) y ofrece una amplia gama de funcionalidades para trabajar con imágenes.
    pip install Pillow
